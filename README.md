### Features
- 🧠 **Persistent Memory** - Remembers your name
- 🗣️ **Voice Output** - Say "speak" to hear Simon  
- 😎 **Personality** - Matches your mood
- 💻 **Code + Teach Mode** - Explains anything

### How to Use
1. `my name is Alex` - Save your name
2. `/help` - See commands
3. `/joke` - Get a joke
4. `explain python and speak` - Get voice
